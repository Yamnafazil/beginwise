Let's Learn the Use of "This" and "That"

What would you say about this cat?

This is used for something that is near you.

That is used for something that is far away.

Examples:

This is a cup. (near)

That is a balloon. (far)

Let's do some Practice Questions:

What would you say about this cat?

This is a cat.
That is a cat.
None of the above.

What would you say about that car?

This is a car.
That is a car.
None of the above.

What would you say about that mountain?

This is a mountain.
That is a mountain.
None of the above.
