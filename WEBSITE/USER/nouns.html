Lets's Learn Nouns, Pronouns, Verbs, and Adjectives

Nouns

A noun is a person, place, or thing. Let's look at some examples.

This is an apple. It is a thing, so it's a noun!

Pronouns

A pronoun is a word that replaces a noun. Instead of saying 'the boy,' we can say 'he'.

This is a boy. We can replace 'the boy' with 'he'.

Verbs

Verbs are action words. Let's see what people are doing.

The boy is running. Running is a verb because it shows what the boy is doing.

Adjectives

An adjective describes a noun. Let's see how things look!

The boy is wearing a big Shirt. The word 'big' describes the Shirt. It's an adjective!
