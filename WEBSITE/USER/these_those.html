Let's Learn This, That, These, Those, and There Are!

What is This? What is That?

This is used to talk about something close. That is used to talk about something far away.

This is a cat. That is a car.

What are These? What are Those?

These is used for multiple things close by. Those is used for multiple things far away.

These are apples. Those are cars.

What is There Are?

There are is used to describe the presence of multiple objects.

There are many books on the shelf.
